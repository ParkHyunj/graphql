# graphql

# 1. Setup

1> npm init -y
=> node repository 초기화
2> npm i apollo-server graphql
=> apollo-server와 graphql 설치  
3> npm i nodemon -D
=> 더 나은 개발을 위해
4> npm run dev 시 nodemon은 작동중이며, server.js를 저장할 때마다 서버를 재시작 시킨다.
5> Error: Apollo Server requires either an existing schema, modules or typeDefs
